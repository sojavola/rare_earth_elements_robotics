## 🎮 **COMMANDES MANUELLES (Alternative)**

Si vous préférez contrôler chaque terminal :

### **Terminal 1 - Serveur Principal**
```bash
source ~/ree_exploration_ros2/install/setup.bash
ros2 run ree_exploration_server server_node
```
### **Terminal 2 - Système IA**
```bash
source ~/ree_exploration_ros2/install/setup.bash
ros2 run generative_ai_layer science_ai_system
```
### **Terminal 3 - Coordinateur**
```bash
source ~/ree_exploration_ros2/install/setup.bash
ros2 run ree_exploration_agent multi_agent_coordinator
```
### **Terminal 4-7 - Robots (4 terminaux)**
```bash
# Terminal 4 - Robot 0
source ~/ree_exploration_ros2/install/setup.bash
ros2 run ree_exploration_agent agent_node 0
# Terminal 5 - Robot 1  
source ~/ree_exploration_ros2/install/setup.bash
ros2 run ree_exploration_agent agent_node 1
# Terminal 6 - Robot 2
source ~/ree_exploration_ros2/install/setup.bash
ros2 run ree_exploration_agent agent_node 2
# Terminal 7 - Robot 3
source ~/ree_exploration_ros2/install/setup.bash
ros2 run ree_exploration_agent agent_node 3
``
### **Terminal 8 - Classificateur**
```bash
source ~/ree_exploration_ros2/install/setup.bash
ros2 run spectral_classifier spectral_classifier_node

```
### **Terminal 9 - Monitoring**
```bash
source ~/ree_exploration_ros2/install/setup.bash
./start_dashboard.sh
```
